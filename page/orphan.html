You must close this tab!

A tab becomes orphaned when two tabs of the same kind (i.e., non-problem tabs, or a tabs for a specific problem) exist simultaneously in a browser. Then the tab create earlier becomes orphaned and can no longer be used.

The only way for there to be two tabs of the same kind is if you open a tab by giving it a URL and a tab of the same kind already exists. Then that previously existing tab will be orphaned.
